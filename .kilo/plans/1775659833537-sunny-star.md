## Implementation Plan

### Phase 1: Font System (Custom Fonts)

**Action**: Integrate provided fonts and create font loading system

**Files to Use:**
- `MetallicSnare/Resources/Scratched Letters.otf` - Title/branding (MK-01)
- `MetallicSnare/Resources/Fresco Stamp.ttf` - UI text, labels, values

**iPlug2 Implementation:**
```cpp
// In config.h - add font file definitions
#define FONT_PRIMARY_FN "Scratched Letters.otf"
#define FONT_SECONDARY_FN "Fresco Stamp.ttf"

// In layout function - load fonts from Resources
void LoadCustomFonts(IGraphics* pGraphics) {
    // Try loading from file first (development)
    bool primaryLoaded = pGraphics->LoadFont("Primary", FONT_PRIMARY_FN);
    bool secondaryLoaded = pGraphics->LoadFont("Secondary", FONT_SECONDARY_FN);
    
    // Fallback to system fonts if custom fonts fail
    if (!primaryLoaded) {
        pGraphics->LoadFont("Primary", "Arial", ETextStyle::Bold);
    }
    if (!secondaryLoaded) {
        pGraphics->LoadFont("Secondary", "Courier New", ETextStyle::Normal);
    }
}
```

**Font Usage Map:**
| Element | Font | Size | Style |
|---------|------|------|-------|
| MK-01 Brand | Scratched Letters | 32px | Normal |
| Parameter Names | Fresco Stamp | 11-12px | Normal |
| Values | Fresco Stamp | 13-14px | Normal |
| Section Labels | Fresco Stamp | 10px | Normal, Uppercase |
| Status Text | Fresco Stamp | 10px | Normal |

**Files to Modify:**
- `config.h` - Add font file name definitions
- `MetallicKnobs.cpp` - Update all IText() calls with new fonts

**Effort**: 2-3 hours
**Impact**: High (brand identity)

### Phase 5: Value Indication & Interactions

**Action**: Add value arcs to knobs and polish interactions

**Value Arc Design:**
- Background arc: `clrBorder` (subtle)
- Value arc: Parameter-specific color
  - SHATTER: `clrAccentRed`
  - TENSION: `clrAccentAmber`
  - PANG: `clrAccentGreen`
  - Others: `clrBorderBright`

**iPlug2 Implementation:**
```cpp
void ClophSVGKnobControl::DrawWidget(IGraphics& g) override {
    const float cx = mWidgetBounds.MW();
    const float cy = mWidgetBounds.MH();
    const float radius = std::min(mWidgetBounds.W(), mWidgetBounds.H()) * 0.5f;
    const float angle = mAngle1 + (static_cast<float>(GetValue()) * (mAngle2 - mAngle1));
    
    // Draw background arc
    g.DrawArc(clrBorder, cx, cy, radius + 8, mAngle1, mAngle2, nullptr, 3.0f);
    
    // Draw value arc
    g.DrawArc(clrValue, cx, cy, radius + 8, mAngle1, angle, nullptr, 3.0f);
    
    // Draw SVG knob
    if (mSVG.IsValid()) {
        g.DrawRotatedSVG(mSVG, cx, cy, radius * 2, radius * 2, angle);
    }
}
```
