# Freeze95

**VST3 for Windows — $X**

[//]: # "Insert screenshot/teaser image URL here"

Three-band stochastic stutter/freeze effect. Splits audio into low/mid/high bands, glitches each one independently with random capture, reverse, slow-mo, and granular modes. Band-aware bit crushing. Host-sync or free-run. Housed in a retro CRT monitor. Zero latency.

## Controls

**CHAOS** 0–100% — stutter intensity from subtle shimmer to full spectral meltdown  
**LO-FI** 0–100% — per-band bit reduction, sample-rate crush, tilt EQ  
**BPM** 20–300 — manual tempo (free-run) or host-synced via **Sync** toggle  
**Power** — bypass

State labels: `TAME → SHIFT → FRACTURE → MELT` / `CLEAN → WARM → GRAIN → CRUSH`

## Requirements

Windows 10/11 64-bit, any VST3 host. ~128 MB RAM. ~2.6 MB download.

## Install

Drop `Freeze95.vst3` into `C:\Program Files\Common Files\VST3\`. Rescan. No installer, no registration.

## Known

Windows VST3 only. No presets (tweak + save in your DAW project).
