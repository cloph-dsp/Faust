## Design Context

### Users
Guitar players, producers, and sound designers using the Grungr stompbox plugin inside a DAW.

### Brand Personality
Gritty, vintage, handmade, utilitarian. The interface should feel like a battered physical pedal, not a polished app.

### Aesthetic Direction
Reference the real Grunge pedal hardware: distressed type, off-center title placement, visible hardware controls, and a worn enclosure with asymmetry and physicality.

### Design Principles
- Preserve the one-row knob layout unless there is a clear hardware reason to move it.
- Use space to create rhythm between the knob band, title band, and stomp area.
- Let the title feel printed onto hardware, not centered like a generic app header.
- Use the secondary font sparingly but visibly so the interface has typographic contrast.
- Favor tactile, readable control groupings over decorative symmetry.
